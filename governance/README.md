# Governance Layer — FBIM TECH

Esta pasta contém a camada de **governança algorítmica** da empresa.

Ela NÃO executa bots.
Ela NÃO cria conteúdo.
Ela NÃO faz trade.

Ela decide:
- quem vive
- quem morre
- quem escala
- quem experimenta

Camadas:
- core → estado e eventos
- hr_bot → ciclo de vida
- finance_bot → risco
- ai_strategist → estratégia
- auto_scale → escala
- lab_bot → experimentos econômicos
